🐍 Python QR Code Generator

A beginner-friendly Python project that generates custom QR codes using the qrcode library. This project allows users to create QR codes for text, links, or any data and save them as image files.

It is a simple yet practical mini project that helps beginners understand how to use external Python libraries and work with image generation.
***
📌 Features

Generate QR codes from any text or URL

Save QR code as a PNG image file

Easy-to-understand and beginner-friendly code

Lightweight and fast execution

Perfect for college mini projects and portfolio
***
🛠️ Technologies Used

Python 3

qrcode library

Pillow (for image processing)

🚀 How to Run the Project

Install the required library:

pip install qrcode[pil]

Run the Python script:

python main.py

The QR code image will be saved in your project folder.
